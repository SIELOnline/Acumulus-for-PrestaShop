Deze Acumulus Customise Invoice module is ontwikkeld door Buro Rader
(http://www.burorader.com/).

Deze module bevat voorbeeldcode die duidelijk maakt hoe gebruik te maken van de
triggers die de Acumulus module definieert en waarmee de factuur aangepast kan
worden direct voordat die naar Acumulus verstuurd wordt, of waarmee de
resulataten van het versturen van de factuur naar Acumulus verwerkt kunnen
worden.

Deze software wordt u "as is" ter beschikking gesteld. Om hier iets zinvols mee
te doen dient de code aangepast te worden, zie hiervoor de commentaren in de
code.

Deze software wordt u ter beschikking gesteld onder de GNU GENERAL PUBLIC
LICENSE versie 3. Voor meer info zie de licentiebestanden die meegeleverd worden
in de zip van deze module.
