Acumulus module hooks
=====================

The Acumulus module defines 3 hooks:

1) actionAcumulusInvoiceCreated
2) actionAcumulusInvoiceSendBefore
3) actionAcumulusInvoiceSendAfter

See the example module acumulus_customise_invoice for usage and documentation.
You can download this example module from the Acumulus forum at
https://forum.acumulus.nl/index.php/topic,2745.0.html.
